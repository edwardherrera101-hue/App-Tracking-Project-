# App-Tracking-Project-

Cycle Compass is a lightweight web app for tracking menstrual cycles. Log the start of your latest period, set your average cycle and period length, and capture wellness notes. Predictions and notes stay stored on your device.

## Run locally

Open `index.html` in your browser, or serve the folder:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
